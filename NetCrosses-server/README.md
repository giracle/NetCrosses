# NetCrosses Server

## Quickstart
1. Install dependencies: `npm install`
2. Build: `npm run build`
3. Start the server: `npm run start:server`
4. Start a test client: `npm run start:client`

## Configuration
- Server config: `config/server.example.toml`
- Client config: `config/client.example.toml`

Copy the example files and replace `token` before running in production.
