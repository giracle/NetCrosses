防火墙助手

生成端口放行命令（仅生成，不自动执行）

当前系统：macOS
# 推荐：系统设置 -> 网络 -> 防火墙 -> 允许传入 # 高级（pf 端口规则，需管理员权限） sudo sh -c 'cat > /etc/pf.anchors/netcrosses <<: "EOF" pass in proto tcp from any to any port {7001,10000:10099} EOF' sudo pfctl -f /etc/pf.conf sudo pfctl -e

提示：命令需要管理员权限，macOS 建议优先通过系统设置允许传入。
