使用方法介绍：
具体使用方法请看client和server中的markdown文件
此处仅介绍如何使用
client：
1. npm install    #(electron下载可能会很慢，需要自行配置镜像站)
2. npm run build
3. npm run dev	
4. 想打包成软件的话，在win中使用命令：  npm run dist:win即可打包成exe文件，详情看client/docs/makrdown文件
server：
在云服务器中运行
1. npm install    #(electron下载可能会很慢，需要自行配置镜像站)
2. npm run build
3. npm run start:server # (建议改用服务方式打开，否则会因为终端关闭导致程序终止)
